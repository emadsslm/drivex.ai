# 🚗 DriveX AI - Smart Driving Assistant PWA

تطبيق ويب تقدمي (PWA) يعمل كمساعد قيادة ذكي لمساعدة السائقين وتوفير تجربة قيادة آمنة وذكية.

## ✨ المميزات
- واجهة مستخدم متجاوبة وسريعة (PWA).
- مساعد ذكي يعتمد على التقنيات الحديثة.
- إمكانية التثبيت على الهواتف الذكية كأنه تطبيق مستقل.

## 🚀 التشغيل المحلي
قم بفتح ملف `index.html` مباشرة في المتصفح أو باستخدام سيرفر محلي (Live Server).
